About This Tool

This tool fetches real-time stock data using the Alpha Vantage API. The API provides:

  • Daily adjusted stock prices
  • 52-week high and low values (calculated from available data)
  • Volume and trading data
  • Historical price information

The default API key has rate limits (5 requests per minute, 500 per day). For unlimited access, get your own free API key from Alpha Vantage.

Example symbols to try: IBM, AAPL, MSFT, GOOGL, AMZN, TSLA, NVDA, JPM

Note: Some API endpoints may return different data structures. This tool handles the "TIME_SERIES_DAILY_ADJUSTED" endpoint specifically.
